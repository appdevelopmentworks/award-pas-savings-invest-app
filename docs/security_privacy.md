# セキュリティ/プライバシー/免責（MVP）
作成日: 2026-01-06

## 1) 取り扱うデータ
- 収入・支出・貯金などの金融情報（個人データ）
- MVPはlocalStorage保存（端末内）

## 2) プライバシー方針（MVP）
- サーバーへ送信しない（分析ツール導入時は明示）
- データ削除ボタンを提供
- 端末共有の注意喚起（家族PC等）

## 3) セキュリティ
- XSS対策（入力値をHTMLに直挿入しない）
- 依存ライブラリの脆弱性チェック（npm audit）
- 将来サーバー保存する場合は暗号化/認証/アクセス制御

## 4) 免責（重要）
- 投資助言ではなく、一般的情報提供
- 将来の成果を保証しない
- 最終判断は利用者
